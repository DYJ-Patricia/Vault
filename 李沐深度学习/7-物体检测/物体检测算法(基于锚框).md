## 兴趣区域（RoI）池化层

- 给定一个锚框，均匀分割成 n×m 块，输出每块里的最大值
- 不管锚框多大，总是输出 nm 个值

**示例**：

- 输入：一个 3×4 的特征图区域
- 操作：2×2 RoI Pooling，将区域划分为 4 块，取每块最大值
- 输出：一个 2×2 的特征图，值为 `[[5, 6], [9, 10]]`


## Fast RCNN

**核心要点**：

- 使用 CNN 对==图片==提取特征，而不是对锚框提取！！！
- 使用 RoI 池化层对每个锚框生成固定长度特征

**流程图**：

1. 输入图像，通过 CNN 提取整张图的特征图。
2. 使用选择性搜索（Selective search）生成候选锚框。
3. 将每个锚框映射到CNN输出的特征图上，通过 RoI 池化层得到固定维度的特征。
4. 经过全连接层（Dense layer）后，分支输出：
    - Category prediction（类别预测）
    - Bounding box prediction（边缘框预测）

## Faster R-CNN

### 速度很慢，应用在对精度特别关心的时候
**核心要点**：

- 使用一个 **区域提议网络（Region Proposal Network, RPN** 来替代启发式搜索，以获得更好的锚框。
- RPN相当于小一点的目标检测算法，提供一堆锚框
**流程图**：

1. 输入图像，通过 CNN 提取整张图的特征图。
2. **区域提议网络（RPN）**：
    
    - 基于特征图，通过卷积层（Conv layer）生成锚框（Anchor box）。
    - 对每个锚框进行二分类（Binary category prediction，判断是否为前景）和边缘框预测（Bounding box prediction）。
    - 经过 NMS 处理，输出高质量的候选区域。
    
3. 将候选区域映射到特征图上，通过 RoI 池化（RoI pooling）得到固定维度的特征。
4. 经过全连接层（Dense layer）后，分支输出：
    - Category prediction（类别预测）
    - Bounding box prediction（边缘框预测）

## Mask R-CNN

- 如果有像素级别的标号，使用 FCN 来利用这些信息
- 无人车这一块用得比较多

**流程图核心模块**：

1. 基础网络：CNN
2. 区域提议网络：Region proposal network
3. 特征对齐层：Rol align
4. 全连接层分支：
    
    - Category prediction（类别预测）
    - Bounding box prediction（边缘框预测）
    
5. 全卷积网络分支：Fully convolutional network → Mask prediction（掩码预测）

![[Pasted image 20260305202609.png]]

### RoI align和 RoI的区别：

- RoI分得不均匀，但是是完整像素块单位
- RoI align分得均匀，分到的像素块的部分需要加权

## 总结

- R-CNN 是最早、也是最有名的一类基于锚框和 CNN 的目标检测算法
- Fast/Faster R-CNN 持续提升性能
- Faster R-CNN 和 Mask R-CNN 是在追求高精度场景下的常用算法

# SSD(单发多框检测)

- Single Shot MultiBox Detector
- Faster R-CNN需要一个主区域一个RPN，相当于做两次预测
- 而SSD一次就过去了


## 生成锚框

- 对每个像素，生成多个以它为中心的锚框
- 给定 n 个大小 s1​,…,sn​ 和 m 个高宽比，那么生成 n+m−1 个锚框，其大小和高宽比分别为：

(s1​,r1​),(s2​,r1​),…,(sn​,r1​),(s1​,r2​),…,(s1​,rm​)

## SSD模型

- 代码中函数class_predictor中的输出：一个高乘宽的input的每一个像素，以其为中心的锚框对于每个类别的分类，即==预测每个锚框的类别==
- bbox_predictor预测每个锚框对真实值的偏移offset

![[Pasted image 20260307231014.png]]

- 一个基础网络来抽取特征，然后多个卷积层块来减半高宽
- 在每段都生成锚框
    - 底部段来拟合小物体，顶部段来拟合大物体
- 对每个锚框预测类别和边缘框
- 没有RPN

![[Pasted image 20260307231321.png]]

- 相对来说精度低，比较快
- 该模型做的比较早，作者后期没做
- 代码实现相对简单，R-CNN非常难

## 代码

- 每个锚框是一个样本，而不是图片

## 总结

- SSD 通过单神经网络来检测模型
- 以每个像素为中心产生多个锚框
- 在多个段的输出上进行多尺度的检测


# YOLO(You Only Look Once)

- 追求速度，非常快
- 工业界用得多

## YOLO（你只看一次）

- SSD 中锚框大量重叠，因此浪费了很多计算
- YOLO 将图片均匀分成 S×S 个网格单元(==每个单元是每一个锚框==)
- 每个网格单元预测 B 个边缘框
- 后续版本（V2, V3, V4…）有持续改进
