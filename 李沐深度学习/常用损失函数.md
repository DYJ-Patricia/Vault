# 均方损失(L2 Loss)

- y为真实值
- y prime为预测值

![[Pasted image 20251122172214.png]]
- 蓝色线为，当y=0时，更换预测值，损失函数的曲线
- 绿色：什么函数（没听清），呈高斯分布
- 橙色：损失的梯度，即导数
- 梯度下降时，根据负梯度方向更新参数，所以导数决定如何更新参数
- 两者离得更远，更新的越频繁

# 绝对值损失函数(L1 Loss)

- 就算隔的很远，权重的更新也不会特别大
- 0点处不可导，且在0点左右在-1和+1摆动，在后期真实值和预测值很接近时不稳定
