# 梯度爆炸的问题

- 值超出值域（infinity）
    - 对于 16 位浮点数尤为严重（数值区间 6e-5 - 6e4）
- 对学习率敏感
    - 如果学习率太大 -> 大参数值 -> 更大的梯度
    - 如果学习率太小 -> 训练无进展
    - 我们可能需要在训练过程不断调整学习率

# 梯度消失的问题

- 梯度值变成 0
    - 对 16 位浮点数尤为严重
- 训练没有进展
    - 不管如何选择学习率
- 对于底部层尤为严重(梯度反传的时候是从上往下的)
    - 仅仅顶部层训练的较好
    - 无法让神经网络更深
